# Limites de Uso por Plano

O uso da API do Scanify é baseado em planos mensais, com número de documentos inclusos:

| Plano       | Documentos/mês | Valor     |
|-------------|----------------|-----------|
| Free        | 50             | R$ 0      |
| Start       | 250            | R$ 96     |
| Grow        | 1000           | R$ 385    |
| Scale       | 2500           | R$ 963    |
| Enterprise  | Personalizado  | Sob consulta |

---

## Excedente

Nos planos pagos, você pode continuar usando a API além do limite contratado.

- Custo por documento excedente: **R$ 0,50**
- Cobrança incluída na próxima fatura

<Warning>
  No plano gratuito, as requisições são bloqueadas ao atingir o limite mensal.
</Warning>
