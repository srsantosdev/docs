O Scanify é uma API que extrai e **valida dados de documentos brasileiros**, como boletos e notas fiscais eletrônicas (NF-e).

Você envia um PDF ou imagem, e recebe os dados estruturados, validados e prontos para automações, IA ou integração com sistemas.
